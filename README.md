# LCS Algorithm Visualizer

**LCS Algorithm Visualizer** is a professional, interactive web application designed to help users understand and explore the Longest Common Subsequence (LCS) and Longest Common Substring algorithms. This tool provides a dynamic visualization of the dynamic programming (DP) table, allowing users to follow each computational step and trace the optimal solution path.

## Key Features

- **Algorithm Selection:** Supports both LCS and Longest Common Substring visualizations.
- **Stepwise Visualization:** Animates the DP table construction, highlighting current cells and dependencies.
- **Playback Controls:**  
  - Play, Pause, Next, and Previous step navigation  
  - Adjustable animation speed  
  - Jump to any step via dropdown
- **Traceback Visualization:** Clearly displays the optimal path for sequence reconstruction.
- **Contextual Explanations:** Provides plain-language explanations for each algorithmic step.
- **Pseudocode Highlighting:** Dynamically highlights relevant pseudocode lines during execution.
- **Complexity Analysis:**  
  - Shows time and space complexity (O(m\*n))  
  - Displays live statistics such as character comparisons and matrix updates
- **Input Flexibility:**  
  - Accepts custom strings or preset examples  
  - Option to toggle case sensitivity
- **Utility Tools:**  
  - Table validation against the correct solution  
  - Export visualization as a PNG image
- **Modern User Interface:**  
  - Responsive layout with dark mode support  
  - Keyboard shortcuts for efficient navigation

## Getting Started

1. Open the [Live Demo](https://longest-common-subsequence-lcs.vercel.app/).
2. Enter two strings or select a preset example.
3. Choose between "Longest Common Subsequence" and "Longest Common Substring" modes.
4. Click **Start** to begin visualization.
5. Use the playback controls to step through the algorithm.
6. Observe the DP table, explanations, and highlighted pseudocode for a comprehensive understanding.
