Introduction

The Longest Common Subsequence (LCS) is a classic problem in computer science, especially in Dynamic Programming. A subsequence is a sequence obtained by deleting zero or more characters from a string without changing the order of the remaining characters. The LCS problem aims to find the longest subsequence that is common to two given strings. Unlike substrings, subsequences do not need to occur contiguously in the original strings. Why is LCS important? It helps measure similarity between two strings. Used in DNA sequence analysis, file comparison tools (like diff), version control systems, and plagiarism detection. Demonstrates the power of dynamic programming, as the problem can’t be solved efficiently using simple recursion due to overlapping subproblems. Key Idea LCS compares two strings character by character and builds a table to store results of subproblems. If characters match → extend the subsequence. If not → take the maximum LCS found by skipping one character from either string.
